#  Advanced Stopwatch Web Application

A modern and responsive Stopwatch Web Application built using HTML, CSS, and JavaScript. This project goes beyond a basic stopwatch by including advanced features such as Lap Recording, Dark/Light Mode, Live Date & Time Display, and Keyboard Shortcut Support.

##  Features

- ▶️ Start Stopwatch
- ⏸️ Pause Stopwatch
- 🔄 Reset Stopwatch
- 🏁 Lap Time Recording
- 🌙 Dark/Light Theme Toggle
- 📅 Live Date & Time Display
- ⌨️ Keyboard Shortcuts
- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
